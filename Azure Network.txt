Network Interface:-
The Network Interface is a Hardware device that is used to route the incoming
and outgoing traffic which we receive from the internet.
All the data are passed in the Network Interface.
The private ip address is used for the internal communication between the Network Interfaces.
whereas public ip address is used for the external communication. The public ip address
is assigned by the azure we dont have the control to assign the public ip address.
it is assigned by the azure.
